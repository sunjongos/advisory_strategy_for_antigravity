# Advisory Strategy for Antigravity 🚀

이 프로젝트는 Antigravity 에이전트 환경 전용으로 설계된 **초고효율 기획/실행 프로토콜 스킬(SKILL)**입니다. 무거운 LLM(Gemini 3.1 Pro / Opus 4.6)의 토큰 낭비를 막고 극강의 병렬 효율을 내기 위해 제작되었습니다.

## 📌 핵심 4대 원칙 (The 4 Golden Rules)
1. **사전 검토 기획 (Pre-flight Advisor & Ultraplan)**
2. **사후 최종 검증 (Post-flight Advisor Check)**
3. **출력 토큰 극강 최적화 (Strict 100 Words Limit)**
4. **고속 병렬 실행 (Fast & Specialized Execution)**

## 🛠 시스템 가동 트리거
* **"ADVISORY STRATEGY 로 하자"** 라는 명령어로 시스템 전역에서 스킬을 즉각 1순위로 가동합니다.

## 📥 설치 방법
본 레포지토리의 `SKILL.md`를 `.gemini/antigravity/skills/advisory-strategy-antigravity/` 폴더 내에 배치하십시오.
